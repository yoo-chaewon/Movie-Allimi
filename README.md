# Movie-Allimi
⏰파이썬으로 영화 예매 오픈 알리미 만들기 ⏰

- 소개

  **파이썬을 이용하여 CGV용산아이파크몰 IMAX영화 예매 오픈 알리미**

  - 파이썬과 requests, bs4 모듈을 통해 CGV상영표 크롤링.
  - 텔레그램 봇을 만들어서 텔레그램 메세지로 알림을 해줄수 있게 함.
  - 스케줄러를 통해 일정 간격마다 실행을 해서 실시간 알리미의 기능을 할 수 있게 함.

  
- 발전시킬 내용
  - 특정 날짜 아닌 지속적으로 사용될 수 있도록 !



- 사용기술

  ```
  ✔️ 웹크롤링
  ✔️ 텔레그램 봇 만들기
  ✔️ 기본적인 EC2서버 구축
  ```
